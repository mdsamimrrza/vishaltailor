# ✂️ New Vishal Tailors - Masterpiece Edition

![Bespoke Tailoring](https://images.unsplash.com/photo-1598257006458-087169a1f08d?q=80&w=2000&auto=format&fit=crop)

A robust, multilingual (English, Hindi, Nepali) monorepo for **New Vishal Tailors**, featuring a dedicated API server and a high-performance Vite-based frontend.

## ✨ Features

- **Heritage Aesthetic**: A premium design that blends tradition with modern web standards.
- **Full Monorepo Structure**: Managed with `pnpm` for efficient development.
- **Dedicated Backend**: Includes an `api-server` for extensible business logic.
- **Comprehensive Sections**: 
  - **The Craft**: Insights into the master's heritage.
  - **Garment Gallery**: Extensive showcase of tailored wear.
  - **Process Walkthrough**: Transparency in the bespoke journey.
  - **Interactive Map**: Precise location tracking in Janakpur Dham.

## 🛠️ Tech Stack

- **Frontend**: React + Vite + Tailwind CSS 4
- **State Management**: TanStack Query
- **Routing**: Wouter
- **Backend**: Node.js / Express (API Server)
- **Package Management**: pnpm

## 🚀 Getting Started

### Prerequisites
- Node.js 20+
- pnpm 10+

### Installation
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Start development (Frontend + API):
   ```bash
   pnpm run dev:all
   ```

## 🌐 Deployment

This project is optimized for **Vercel**. 
- The root `package.json` contains the necessary build scripts for Vercel.
- Simply connect the repository and Vercel will handle the monorepo deployment.
